# Amazon Review Sentiment Analysis

A Deep Learning NLP application that predicts whether an Amazon product review expresses **Positive** or **Negative** sentiment.

## Features
- Predicts sentiment from Amazon product reviews
- Deep Learning model built with TensorFlow & Keras
- Text preprocessing using Tokenizer and Padding
- Interactive Streamlit web application
- Simple and user-friendly interface

## Tech Stack
- Python
- TensorFlow
- Keras
- Streamlit
- NumPy
- Pandas
- Scikit-learn

## Project Structure
```
amazon-review-sentiment-analysis/
│── app.py
│── sentiment_model.keras
│── tokenizer.pkl
│── config.json
│── requirements.txt
│── README.md
```

## Dataset
- Amazon Product Reviews Dataset
- Binary Sentiment Classification (Positive / Negative)

## Run Locally

1. Clone the repository
```bash
git clone https://github.com/your-username/Product-Review-Sentiment-Analysis.git
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run the application
```bash
streamlit run app.py
```
